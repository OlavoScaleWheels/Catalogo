Como ler o código das rodas

Formato geral: MMM.CCC.TT

  • MMM = Modelo (3 dígitos). Ex.: 031 → modelo 031.
  • CCC = Cor (3 dígitos). Ex.: 001 = Cromado, 002 = Dourado, 003 = Preto, 020 = Azul.
  • TT = Tamanho. Pode ter ponto ou vírgula. Ex.: 11, 10, 12,5.
Exemplos
  • 031.001.11 → Modelo 031, Cor 001 (Cromado), Tamanho 11
  • 031.002.11 → Modelo 031, Cor 002 (Dourado), Tamanho 11
  • 031.003.10 → Modelo 031, Cor 003 (Preto), Tamanho 10
  • 031.020.12,5 → Modelo 031, Cor 020 (Azul), Tamanho 12,5
